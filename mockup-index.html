Self-Build Navigator

A thoughtfully designed project management platform for UK self-builders. These mockups demonstrate the calm, professional aesthetic and user-friendly interface.

Design Mockups — November 2025

Dashboard View

Main overview showing project stats, high-priority tasks, and upcoming deadlines at a glance.

  • Stats cards (Total, In Progress, Completed, Blocked)
  • Overall progress visualization
  • High priority task list
  • Upcoming due dates with status indicators
  • Expandable sidebar navigation
  • Global search with keyboard shortcut

Timeline View

Gantt-style timeline showing task schedules, durations, and progress across the project.

  • Gantt chart with horizontal progress bars
  • Color-coded by status (in-progress, blocked, review, done)
  • Percentage completion indicators
  • Collapsed sidebar for maximum timeline visibility
  • Task status tags and priority levels
  • Responsive layout (minimum 900px width)
